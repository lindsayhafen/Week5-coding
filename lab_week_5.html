Summary stats, check for equal variances here too

-Starting from penguins_f, create a summary table that groups by species and returns: -mean of flipper length -sd of flipper_length_mm -sample size -variance var -standard error = sd/sqrt(n)

  • Guideline: if your largest var for a group is less than 4x greater than smallest var for a group, then var is close enough to perform ANOVA
penguins_f_summary <- penguins_f %>% 
  group_by(species) %>% 
  summarize(mean_flipper_length = mean(flipper_length_mm),
            sd_flipper_length = sd(flipper_length_mm),
            sample_size = n(),
            variance = var(flipper_length_mm),
            standard_error = sd(flipper_length_mm)/sqrt(n()),
            var_flipper_length = var(flipper_length_mm))

Lavenes - if n() is great enough, you will get a significant difference no matter what, so be careful to look and think and check guideline - 0.05782 Pr number is greater than our usual significance value of .05, so we will retain the null that the variance is equal

How to make a Beeswarm plot

  • Show some summary statistics
  • Mean =/- 1 sd
  • Generic plot is better than our modificaitons, modifications for example
  • Add geom points to layer graphs on top of each other.
ggplot() +
  geom_beeswarm(data = penguins_f, 
                aes(x = species, y = flipper_length_mm),
                color = "darkturquoise",
                size = 3,
                alpha = 0.5) +
  geom_point(data = penguins_f_summary, aes(x = species, y = mean_flipper_length),
             size = 4) +
  geom_errorbar(data = penguins_f_summary,
                aes(x = species,
                    ymax = mean_flipper_length + sd_flipper_length,
                    ymin = mean_flipper_length - sd_flipper_length),
                width = 0.1)

One-way ANOVA

  • Start by aov -> runs type 1 ANOVA (If everything is matched (balanced design, equal sample size etc, then its the same as a type III anova)) F statistic is the ratio of the between group variance divided by the within group variance (Variance = sum of squares)
  • 2e-16*** is the smallest number that R will display (Super small propbability that the populations are all from a population that has the same mean)
penguin_aov <- aov(flipper_length_mm ~ species, data = penguins_f)
summary(penguin_aov)
##              Df Sum Sq Mean Sq F value Pr(>F)    
## species       2  21416   10708   411.8 <2e-16 ***
## Residuals   162   4213      26                   
## ---
## Signif. codes:  0 '***' 0.001 '**' 0.01 '*' 0.05 '.' 0.1 ' ' 1
TukeyHSD(penguin_aov)
##   Tukey multiple comparisons of means
##     95% family-wise confidence level
## 
## Fit: aov(formula = flipper_length_mm ~ species, data = penguins_f)
## 
## $species
##                       diff       lwr       upr     p adj
## Chinstrap-Adelie  3.940774  1.436275  6.445272 0.0007922
## Gentoo-Adelie    24.912376 22.790647 27.034105 0.0000000
## Gentoo-Chinstrap 20.971602 18.366227 23.576977 0.0000000
  • All of the pairwise adjusted values from Tukeys are below .05, so the post-hoc test tells us that there is a significant difference between each combination of the groups.

If you have very different sample sizes, unbalanced designs, differences between groups and distributions etc then do a type III ANOVA with the car package:

-lm stands for linear model

penguin_lm <- lm(flipper_length_mm ~ species, data = penguins_f)
